# 伯爵山莊 · 完整社區情報站

> 一個伯爵山莊，安放每一種生活。

汐止伯爵山莊的完整社區情報站 — 社區歷史、在售物件、周邊地圖、日照與微氣候資料，一個網址看清。

**🔗 線上網站：** https://bojue-estate.vercel.app/

---

## 關於這個網站

由台灣房屋東湖特許加盟店 · 晟華製作，主打「誠實、可查證的專業」。核心理念延續仲介心骨 —— **對象是人，不是物件**。帶看前先讓客戶把社區看清楚，比口頭說明更有說服力。

## 功能

- **社區四十年 history** — 從張姚宏影（張媽媽）民國 68 年造鎮，到日月光半導體創業基石，再到湖興里成立
- **8 筆在售物件** — 伯爵一代透天、仁愛特區 ×3、伯爵夫人 ×2、建業樓、青春嶺，各附完整規格與特色
- **周邊戰情地圖** — Leaflet 地圖，16 個社區聚落範圍、590／F909 公車路線、生活機能，圖層可開關
- **日照與太陽軌跡** — SunCalc 實算（緯度 25.08°N），四季太陽走位，證明朝南棟別採光優勢
- **全年微氣候時鐘** — 中央氣象署汐止站（C0AH00）2012–2025 十四年實測。點月份，地圖疊出雨量色罩與風向箭頭，依四季配色
- **專業工具** — 格局神器、頭期款計算機

## 技術

- 單頁 HTML / CSS / Vanilla JS
- Leaflet（地圖）、SunCalc（日照計算）
- 資料來源：中央氣象署 C0AH00 氣象資料、實價登錄、政府謄本、現場勘查
- 部署：GitHub + Vercel 自動部署

## 資料來源與免責

本資訊由台灣房屋・永悅團隊整理，資料來源為政府謄本、實價登錄、地政與氣象公開資料及現場勘查。氣候數值來自中央氣象署汐止測站（C0AH00）2012–2025 年逐月觀測平均；風向為風速加權向量平均，本地數值受山谷地形影響，與綜觀尺度季風方向可能不同，僅供居住參考。所列數值僅供參考，實際條件以正式簽約文件為準。
